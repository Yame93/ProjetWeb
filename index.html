Projet DEV.Web

Le But de ce projet?
Le Projet Web a pour but de nous former a crée un site internet qui pourra nous aider a crée le notre dans le futur comme pour notre portfolio

les 4 Membres:

leurs objectifs

Page d'Accueil

il a pour mission de cree une page d'accueil qui presente le projet et toute les missions et les personnes qui font les missions, pour finir il doit faire une conclusion du projet

mission:1

Il a pour mission de cree un bilan structuré sur l'étude des données de KDou puis trouver des perspectives d'améliorations,tout sa dans une nouvelle page web

Mission:2

il a pour mission d'exploiter une base de donnée grace a une interface de type "low code" appeller QBE (Querry by example),tout sa dans une nouvelle page web

Mission:3

il a pour mission d'Analyse des données de KDou avec le langage SQL, puis de présenter 2 indicateurs qui sont intéressants de chaque thèmes,enfin il doit présenter tout c'est indiquateurs et mettre tout sa dans une nouvelle page web

CONCLUSION

Toutes ces mission donnent à KDou les informations nécessaires pour prendre des décisions claires et ajuster sa stratégie commerciale en utilisant les informations. La première mission permet de définir la stratégie globale de repositionnement "éco-friendly", la deuxième mission fournit des outils pour extraire et comprendre les données clés de l'entreprise, et la troisième mission permet de plonger plus en profondeur dans l’analyse des données via SQL. Ces missions devraient aider KDou à améliorer sa compétitivité tout en répondant à une demande croissante pour des produits respectueux de l'environnement.
